## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`MONGOURI`
`JWT_SECRET`
`ADD_KEY=1981`
`EMAIL_PASS`
`EMAIL_USER`
`WEBSITE`
`WEBSITE_NAME`
`WEBSITE_EMAIL`
`CLOUDINARY_CLOUD_NAME`
`CLOUDINARY_API_KEY`
`CLOUDINARY_API_SECRET`
`RAZOR_KEY_ID`
`RAZOR_KEY_SECRET`
